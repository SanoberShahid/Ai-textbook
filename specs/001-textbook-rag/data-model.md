This document describes the key data entities for the project.

## Entities

### Chapter

- **Description**: A discrete section of the textbook content.
- **Fields**:
    - `id`: Unique identifier for the chapter (e.g., `chapter-1`).
    - `title`: The title of the chapter.
    - `content`: The full content of the chapter in Markdown format.

### Textbook Document

- **Description**: A chunk of the textbook content, prepared for the RAG system. This is a derived entity, created by splitting the `Chapter` content.
- **Fields**:
    - `id`: Unique identifier for the document chunk.
    - `chapter_id`: The identifier of the chapter this chunk belongs to.
    - `text`: The text content of the chunk.
    - `embedding`: The vector embedding of the `text`.

### User Query

- **Description**: Text input from the user for the RAG chatbot.
- **Fields**:
    - `query_text`: The user's question.
    - `timestamp`: The time the query was submitted.

### AI Response

- **Description**: Text output from the RAG chatbot.
- **Fields**:
    - `response_text`: The answer generated by the chatbot.
    - `source_documents`: A list of `Textbook Document` IDs that were used to generate the answer.
    - `timestamp`: The time the response was generated.
